Socbib

An intermittently-updated BibTeX file for sociologists. Mostly reflects the interests of the maintainer: Economic Sociology, Sociology of Culture, Social Theory, IT, Blood/Organ donation, human goods, Open Source Software. Contributions welcome.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/kjhealy/socbibs
